# Spring Boot Real-Time Chat Application

https://ganapatirupti.github.io/springboot-chat-application/

A real-time chat application built using Spring Boot and WebSocket.

## Features
- Real-time messaging using WebSocket
- STOMP protocol with SockJS
- Thymeleaf-based UI
- Bootstrap for responsive design

## Tech Stack
- Java
- Spring Boot
- WebSocket (STOMP, SockJS)
- Thymeleaf
- Bootstrap

## How to Run
1. Clone the repository
2. Run the Spring Boot application
3. Open browser and access the application on configured port
